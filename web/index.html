How to set up the bot

Note: In the context of this bot, the words guild and server are both synonyms for a discord guild / server.

First invite the bot to your server by clicking here!
Create at least one voice channel in any category on your server and join the channel once.
Now the guild should appear in the sidebar of this website and the category is listed for configuration.

To display additional categories in the configurator, simply create a voice channel again and connect it once.
