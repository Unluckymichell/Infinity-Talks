Infinity Talks Config 

Configuring Infinity Talks Bot for server 

Language

Language for the selected guild. Applies to bot commands and all Discord features.

Prefix 

Prefix for the selected guild. A trailing space is allowed. The prefix is recognized exactly as it was written before command.

Prefix can not be empty!

Categorys

There are no categories in the database! New categories are only added if there is at least one voice channel and it has been joined once.

Categorys

Select a category to modify. You can have multiple categorys with Infinity Talks enabled and each one has independent settings!
New categories are only added if there is at least one voice channel and it has been joined once.

Enable

Activates the Infinity Talks functionality.

Allow Lock

Allow users to lock their voice channels.

Channel Limit

Limit the number of channels the bot creates in a category (0 for no limit).

Only positive values allowed!

Channel User Limit

Set the default user limit for unlocked channels (0 for no limit).

Values from 0 to 99 allowed!

Naming Rule

Used to name the automatically generated channels.
